## Quick start üöÄ

### Requirements
- [pnpm](https://pnpm.io/) (Fast, disk space efficient package manager)
- [NodeJS 23+](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
### Library installation
#### core
```bash
pnpm add @zap/flash
```
#### langchain
```bash
pnpm add @zap/flash-langchain
```
